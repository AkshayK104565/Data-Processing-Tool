How to use Pattern Data Processing Automation Tool — quick guide for users and admins.
For users — running a tool
1
Find the tool using search or category filter tabs.
2
Click ↓ Template to download the Excel template.
3
Fill in data following the on-screen instructions inside the Run panel.
4
Click Run →, upload your filled file, then hit Process & Download.
5
The output file downloads automatically with all original sheets preserved, plus a new output sheet.
Pattern-based tools — important notice
!
These tools rely on recognising specific data patterns. If the source data format changes, the tool may produce incomplete or incorrect results.
!
Always cross-check the output against the source data when running a tool for the first time or after a format update.
For admins — adding a tool
1
Upload the .xlsx template to Google Drive / SharePoint and copy the direct-download link.
2
Click + Add Tool, fill in all details including step-by-step instructions.
3
Click Add to Hub — the card appears immediately.
